# @holochain-open-dev/notifications

This module follows the [holochain-open-dev](https://github.com/holochain-open-dev/) pattern of developing holochain modules. Read [its documentation](https://holochain-open-dev.github.io) to understand more about its motivation and the big picture.

What it does:

This module allows holochain applications to send notifications to text, whatsapp and email via a dedicated notifier agent, which stores contact information and handles notification requests, or "tips".
